## Key Decisions & Rationale

- **Window size increased to 200 kb**: reduces noise in the methylation Manhattan plot, improves syntenic window correlation (r 0.77 → 0.89).
- **Coverage check before biological interpretation**: confirmed male intermediate sites are not a sequencing depth artefact before drawing biological conclusions.
- **Genomic context used EDTA GFF3 major classes**: LTR, DNA, LINE, MITE, pararetrovirus, Unknown (same classification scheme as landscape figures). Gene body = entire gene feature from BRAKER3 GTF (exons + introns combined).

## Technical Details

### `papro_methyl_context.py` — key functions
- `load_meth()`: loads modkit bedMethyl, filters on mod type `m` and min coverage, returns sorted (pos, cov, pct) arrays per scaffold
- `label_sites()`: uses `bisect.bisect_left/right` to assign each site to gene body and/or repeat major class in O(n_intervals × avg_interval_width) — fast for 13M sites
- `categorize()`: bins pct array into unmethylated/intermediate/methylated
- `context_fracs()`: computes mutually exclusive context fractions (intergenic / gene_only / gene+repeat / repeat_only by major class)
- Figure: 2-panel — top row coverage violins (log₁₀ scale), bottom row stacked context bars

### Bisect-based interval labeling
```python
def label_sites(pos_arr, gene_ivs, rep_ivs_by_major):
    n = len(pos_arr)
    is_gene = np.zeros(n, dtype=bool)
    rep_cls = np.full(n, '', dtype=object)
    for major, ivs in rep_ivs_by_major.items():
        for s, e in ivs:
            lo = bisect.bisect_left(pos_arr, s)
            hi = bisect.bisect_right(pos_arr, e)
            if lo < hi:
                rep_cls[lo:hi] = major   # last-write wins for overlapping intervals
    for s, e in gene_ivs:
        lo = bisect.bisect_left(pos_arr, s)
        hi = bisect.bisect_right(pos_arr, e)
        if lo < hi:
            is_gene[lo:hi] = True
    return is_gene, rep_cls
```

## Biological Interpretation Written Up

A full scientific write-up of the methylation findings was produced (suitable for results/supplementary), covering:
1. Overall sex difference in autosomal 5mC (59.7% F vs 37.0% M)
2. Coverage analysis ruling out technical noise for male intermediate sites
3. LTR transposons as the dominant context for both intermediate (male) and fully methylated (female) sites
4. Gene bodies CpG-hypomethylated in both sexes
5. Hemi-methylation of male LTR elements as the most parsimonious explanation (replication-associated hemi-methylation in mitotically active antheridial tissue, vs. genuine sex-specific difference in TE silencing capacity)
